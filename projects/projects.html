Temperature Controller with On-Off & PID (Graduation Project)

Accurate temperature control is essential in modern industrial applications. This project implements a closed-loop system using three control strategies: PID, On/Off, and Slow PWM. Users can monitor real-time data, switch between control modes, and adjust parameters through an interactive interface — making the system ideal for both testing and educational purposes.

The system continuously reads temperature data and processes it in real time. Based on the selected control strategy, it dynamically adjusts the heating output. Users can set a target temperature, choose a control mode, and fine-tune relevant parameters through a user-friendly interface. Each control mode is designed to demonstrate how different algorithms respond under identical thermal conditions.
